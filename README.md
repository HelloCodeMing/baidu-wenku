# 描述
  去掉百度文库的浮动广告、冗余元素，使得整个页面可以用 Ctrl+P 来打印成pdf文档
  
  
# 使用方法
- 复制index.js里的脚本内容，然后打开浏览器(比如Chrome)的控制台，粘贴并执行；
- 在弹出的打印窗口，优先选择“输出到PDF文件”，打印即可；
- 也可以取消打印窗口，将网页另存为mhtml格式;
